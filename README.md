# 酒单管理系统-管理端

vue3 + element-ui

## 技术栈
- Vue 
- Vue Router
- Vuex 
- Vue CLI
- Element-UI-Plus
- Sass 
- ES6
- Axios


## 安装与使用
```shell
# 安装依赖
npm install

# 本地运行
npm run serve 	

# 打包
npm run build		
```

## 说明
该后台系统采用vue动态路由来管理权限，分为三个权限：普通、高级、超级；
不同权限所能使用的功能如下：
- 普通管理员：订单管理
- 高级管理员：订单管理、房间管理
- 超级管理员：订单管理、房间管理、用户管理
